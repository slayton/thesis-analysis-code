remove_callback

synopsis

removes callbacks from object

syntax

    REMOVE_CALLBACK(hObj, callback_name)
    REMOVE_CALLBACK(hObj, callback_name, callback_id)
    n=REMOVE_CALLBACK(...)

description

REMOVE_CALLBACK(hObj, callback_name) removes all callback functions from the object with handle hObj.

REMOVE_CALLBACK(hObj, callback_name, callback_id) removes all callback functions with the specified callback ID from the object with handle hObj.

n=REMOVE_CALLBACK(...) returns the number of callback functions removed

example

hFig = figure;
event_dispatch( hFig );
fcn = @(hObj, eventdata) fprintf( 'Wheel moved\n' );
fcn_id = add_callback( hFig, 'MyWheelMovedFcn', fcn );
remove_callback( hFig, 'MyWheelMoved', fcn_id );
