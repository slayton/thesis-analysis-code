process_eeg_clip

synopsis

find out of range values in eeg data

syntax

    PROCESS_EEG_CLIP(time,data,expansion)

description

PROCESS_EEG_CLIP(time,data,expansion) returns all time segments when the eeg data has been clipped (i.e. abs( data ) >= 2047). The expansion argument specifies how many samples on either side of clipped values will be included as well (default=5).
