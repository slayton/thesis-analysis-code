isolation_distance

synopsis

compute isolation distance measure

syntax

    d=ISOLATION_DISTANCE(distances,idx)

description

d=ISOLATION_DISTANCE(distances,idx) where distances is a vector of mahalanobis distances between feature vectors and the centroid of a subset (i.e. cluster) of feature vectors (given by the indices idx). Returns the isolation distance between the cluster and the remaining spikes. if idx is a cell array of indices for c different clusters, then distances should be a n-by-c matrix.
