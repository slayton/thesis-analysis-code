seg_sort

synopsis

sort a list of segments on the segment start time

syntax

    seg=SEG_SORT(segments)
    [seg,i]=SEG_SORT(segments)

description

seg=SEG_SORT(segments) sorts segments based on start time

[seg,i]=SEG_SORT(segments) also returns the indices of the sorted segments into the original segment list. I.e. seg=segments(i,:).
