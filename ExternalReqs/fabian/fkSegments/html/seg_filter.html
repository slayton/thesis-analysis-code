seg_filter

synopsis

select segments

syntax

    seg=SEG_FILTER(segments,events)
    seg=SEG_FILTER(segments,events,nmin)
    seg=SEG_FILTER(segments,events,nmin,nmax)
    [seg,i]=SEG_FILTER(...)

description

seg=SEG_FILTER(segments,events) returns all segments that contain at least one event.

seg=SEG_FILTER(segments,events,nmin) returns all segments that contains at least nmin events.

seg=SEG_FILTER(segments,events,nmin,nmax) returns all segments that contains at least nmin and at most nmax events.

[seg,i]=SEG_FILTER(...) also returns the indices of the returned segments in the original segments list. I.e. seg=segments(i,:).
