segcount

synopsis

cumulative count of segments

syntax

    fcn=SEGCOUNT(segments)
    c=SEGCOUNT(segments,x)

description

fcn=SEGCOUNT(segments) returns a handle to a function that accepts a single argument and returns the cumulative count of segments at the values in the argument. (i.e. the function counts the number of segments which contain the value of the argument).

c=SEGCOUNT(segments,x) returns the count of segments at x.
