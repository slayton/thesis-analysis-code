seg_overlap

synopsis

compute overlap between segments

syntax

    [overlap,rel1,rel2]=SEG_OVERLAP(a)
    [overlap,rel1,rel2]=SEG_OVERLAP(a,b)

description

[overlap,rel1,rel2]=SEG_OVERLAP(a) returns the absolute overlap and overlap fraction (relative to length of first or second segment) between all combinations of segments in a.

[overlap,rel1,rel2]=SEG_OVERLAP(a,b) returns the absolute overlap and overlap fraction (relative to length of first or second segment) between all combinations of segments in a and b.
