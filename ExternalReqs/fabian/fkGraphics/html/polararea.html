polararea

synopsis

creates a polar area plot

syntax

    h=POLARAREA
    h=POLARAREA(rho)
    h=POLARAREA(theta,rho)
    h=POLARAREA(hax,...)
    h=POLARAREA(...,param1,val1,...)

description

h=POLARAREA plots a polar area plot of the function |sin(2*theta)|. A handle to a polar area graphics object is returned.

h=POLARAREA(rho) plots a polar area of radius data at equally spaced angles. If rho is a matrix, multiple area plots will be created.

h=POLARAREA(theta,rho) plots a polar area in the current axes using the angle and radius data. Theta and rho can be either vectors or matrices of the same size. Alternatively, theta can be a vector and rho can be a matrix with the same number of rows as the length of theta. In this case the function will create as many polar area plots as there are columns in rho.

h=POLARAREA(hax,...) will plot in axes with handle hax.

h=POLARAREA(...,param1,val1,...) sets polar area properties through parameter/value pairs. Execute set(h) to see a list of valid properties that can be modified.
