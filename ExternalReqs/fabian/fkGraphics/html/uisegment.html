uisegment

synopsis

gui to define segments

syntax

    S=UISEGMENT()
    S=UISEGMENT(X,Y,S)

description

S=UISEGMENT() plots vector Y vs X in a slider plot, lets the user define segments and returns the start and end times of each segment in the nx2 matrix S. Middle mouse button adds a segment, right mouse button deletes a segment.

S=UISEGMENT(X,Y,S) takes a nx2 matrix of initial segments
