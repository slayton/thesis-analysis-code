circle

synopsis

draw a circle

syntax

    h=CIRCLE(center,radius)
    h=CIRCLE(hax,...)
    h=CIRCLE(...,param1,val1,...)

description

h=CIRCLE(center,radius) draws a circle with specified center and radius in current axes

h=CIRCLE(hax,...) draws circle in specified axes

h=CIRCLE(...,param1,val1,...) draws circle with additional parameter/value pairs, which should be valid parameters for rectangle function.
