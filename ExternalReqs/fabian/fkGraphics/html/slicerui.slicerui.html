slicerui

synopsis

slicer ui constructor

syntax

    slicer=SLICERUI
    slicer=SLICERUI(slicer)
    slicer=SLICERUI(grid)
    slicer=SLICERUI(grid,state)

description

slicer=SLICERUI default constructor, create empty slicer ui object.

slicer=SLICERUI(slicer) copy constructor

slicer=SLICERUI(grid) create new slicer ui object and set the initial grid.

slicer=SLICERUI(grid,state) sets the initial state.
