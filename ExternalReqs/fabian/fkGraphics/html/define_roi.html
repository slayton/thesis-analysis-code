define_roi

synopsis

define named regions of interest

syntax

    regions=DEFINE_ROI(parm1,val1,...)

description

regions=DEFINE_ROI(parm1,val1,...) This function will present a gui in which rectangles can be drawn to define named regions of interest. Valid options are:

image
video image (default=[])
imagesize
2x2 matrix with x and y size of image (default=[1 size(image,2); 1 size(image,1)])
regions
regions structure array, with fields 'name' and 'position'
