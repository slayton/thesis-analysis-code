polarline

synopsis

creates a polar line plot

syntax

    h=POLARLINE
    h=POLARLINE(rho)
    h=POLARLINE(theta,rho)
    h=POLARLINE(hax,...)
    h=POLARLINE(...,param1,val1,...)

description

h=POLARLINE plots a polar line plot of a von Mises distribution. A handle to a polar line graphics object is returned.

h=POLARLINE(rho) plots a polar line of radius data at equally spaced angles. If rho is a matrix, multiple area plots will be created.

h=POLARLINE(theta,rho) plots a polar line in the current axes using the angle and radius data. Theta and rho can be either vectors or matrices of the same size. Alternatively, theta can be a vector and rho can be a matrix with the same number of rows as the length of theta. In this case the function will create as many polar line plots as there are columns in rho.

h=POLARLINE(hax,...) will plot in axes with handle hax.

h=POLARLINE(...,param1,val1,...) sets polar line properties through parameter/value pairs. Execute set(h) to see a list of valid properties that can be modified.
