plotpolyline

synopsis

plot a polyline

syntax

    h=PLOTPOLYLINE(nodes)
    h=PLOTPOLYLINE(struct)
    h=PLOTPOLYLINE(...,arg1,arg2,...)

description

h=PLOTPOLYLINE(nodes) plots a polyline defined by the nx2 matrix and returns a handle.

h=PLOTPOLYLINE(struct) plots a polyline defined by the structure.

h=PLOTPOLYLINE(...,arg1,arg2,...) passes extra arguments to the line function that is used to plot the polyline.
