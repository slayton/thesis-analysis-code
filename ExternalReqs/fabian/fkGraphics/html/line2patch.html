line2patch

synopsis

create patches along a line

syntax

    [px,py]=LINE2PATCH(x,y)
    [px,py]=LINE2PATCH(x,y,width)

description

[px,py]=LINE2PATCH(x,y) construct patches for a polyline with x,y coordinates.

[px,py]=LINE2PATCH(x,y,width) set the width of the line.
