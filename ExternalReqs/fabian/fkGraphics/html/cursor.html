cursor

synopsis

creates a cursor

syntax

    h=CURSOR
    h=CURSOR(x,y)
    h=CURSOR(hax,...)
    h=CURSOR(...,param1,val1,...)

description

h=CURSOR creates a new cursor in the center of the current axes. A handle to the cursor is returned.

h=CURSOR(x,y) creates a new cursor at coordinates x,y in the current axes.

h=CURSOR(hax,...) creates a cursor in the axes with handle hax.

h=CURSOR(...,param1,val1,...) sets cursor properties through parameter/value pairs. Execute set(h) to see a list of valid properties that can be set.
