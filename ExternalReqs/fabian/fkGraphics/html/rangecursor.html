rangecursor

synopsis

creates a cursor

syntax

    h=RANGECURSOR
    h=RANGECURSOR(xrange,yrange)
    h=RANGECURSOR(hax,...)
    h=RANGECURSOR(...,param1,val1,...)

description

h=RANGECURSOR creates a new range cursor in the center of the current axes. A handle to the cursor is returned.

h=RANGECURSOR(xrange,yrange) creates a new range cursor with specified x and y ranges in the current axes.

h=RANGECURSOR(hax,...) creates a cursor in the axes with handle hax.

h=RANGECURSOR(...,param1,val1,...) sets range cursor properties through parameter/value pairs. Execute set(h) to see a list of valid properties that can be set.
