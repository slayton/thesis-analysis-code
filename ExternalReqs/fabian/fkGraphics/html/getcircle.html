getcircle

synopsis

let's you define a new circle

syntax

    [center,radius]=GETCIRCLE
    [center,radius]=GETCIRCLE('axes',hAx)

description

[center,radius]=GETCIRCLE opens a new figure and axes in which you can select a circle. Left mouse button will start the circle and dragging the mouse will increase/decrease the radius. Releasing the mouse button finishes the circle. Pressing 'Esc' will abort. The function returns the center and the radius of the selected circle.

[center,radius]=GETCIRCLE('axes',hAx) specifies the handle of the parent axes.
