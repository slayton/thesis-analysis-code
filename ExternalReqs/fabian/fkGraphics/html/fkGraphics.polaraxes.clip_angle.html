clip_angle

synopsis

clip angles

syntax

    t=CLIP_ANGLE(h,angles)
    t=CLIP_ANGLE(h,angles,clip)
    t=CLIP_ANGLE(h,angles,clip,limits)

description

t=CLIP_ANGLE(h,angles) clip angles to angular limits of polar axes h. Values outside the limits are set to NaN.

t=CLIP_ANGLE(h,angles,clip) specifies the clip method:

nan
clipped values are set to NaN
clip
clipped values are set to angular limits

t=CLIP_ANGLE(h,angles,clip,limits) uses the specified angular limits rather than the limits of the polar axes.
