slider

synopsis

slider object constructor

syntax

    s=SLIDER
    s=SLIDER(s)
    s=SLIDER(h)
    s=SLIDER(h,parm1,val1,...)

description

s=SLIDER default constructor, creates an empty slider object

s=SLIDER(s) copy constructor

s=SLIDER(h) create slider in container with handle h.

s=SLIDER(h,parm1,val1,...) set optional parameters. Valid parameters are: limits, center, windowsize, updatemode, color
