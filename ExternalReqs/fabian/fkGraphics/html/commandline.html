commandline

synopsis

custom command line in figure

syntax

    COMMANDLINE
    COMMANDLINE(h)
    COMMANDLINE(h,fcn)

description

COMMANDLINE create a command line in a new figure. All commands are executed in the base workspace.

COMMANDLINE(h) create command line in figure or uipanel with handle h.

COMMANDLINE(h,fcn) the command is passed to the custom function. The function should return a character array with the result that will be displayed.
