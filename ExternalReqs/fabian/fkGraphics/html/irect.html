irect

synopsis

draw an editable rectangle

syntax

    h=IRECT(position)
    h=IRECT(...,parm1,val1,...)

description

h=IRECT(position) draws a rectangle defined by the position vector ([left bottom width height]) and allows the user to interactively change the circle. The function returns a handle to the rectangle. Get(h,'rect') will return a structure with rectangle data. Dragging the sides aor corners will change the size of the rectangle, dragging the center point, will move the rectangle.

h=IRECT(...,parm1,val1,...) passes in extra parameter/value pairs. Valid parameters are:

axes
handle of parent axes
need_selection
rectangle can be edited only when it is selected
selected
0/1 selected state of rectangle
