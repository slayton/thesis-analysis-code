javacommandline

synopsis

custom command line (java component)

syntax

    [jc,hc]=JAVACOMMANDLINE
    [...]=JAVACOMMANDLINE(h)
    [...]=JAVACOMMANDLINE(h,fcn)

description

[jc,hc]=JAVACOMMANDLINE creates a new command line and returns the handle of the java component jc and the handle of the graphics container hc. Commands are executed in the base workspace.

[...]=JAVACOMMANDLINE(h) create command line in figure with handle h.

[...]=JAVACOMMANDLINE(h,fcn) the command is passed to the custom function. The function should return a character array with the result that will be displayed.
