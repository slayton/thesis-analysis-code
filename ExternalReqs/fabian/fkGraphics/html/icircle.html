icircle

synopsis

draw an editable circle

syntax

    h=ICIRCLE(center,radius)
    h=ICIRCLE(...,parm1,val1,...)

description

h=ICIRCLE(center,radius) draws a circle defined by center and radius and allows the user to interactively change the circle. The function returns a handle to the circle. Get(h,'circle') will return a structure with circle data. Dragging the center point of the circle will move the circle, dragging the circle itself will change its radius.

h=ICIRCLE(...,parm1,val1,...) passes in extra parameter/value pairs. Valid paramters are:

axes
handle of parent axes
need_selection
circle can be edited only when it is selected
selected
0/1 selected state of circle
