plotcircle

synopsis

plot a circle

syntax

    h=PLOTCIRCLE(center,radius)
    h=PLOTCIRCLE(...,arg1,arg2,...)

description

h=PLOTCIRCLE(center,radius) plots a circle defined by the center and radius and returns a handle.

h=PLOTCIRCLE(...,arg1,arg2,...) passes extra arguments to the rectangle function that is used to plot the circle.
