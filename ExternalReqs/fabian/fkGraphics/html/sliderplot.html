sliderplot

synopsis

simple plot with slider

syntax

    [hax,hs]=SLIDERPLOT(...)
    [hax,hs]=SLIDERPLOT({...},{...},...)
    [hax,hs]=SLIDERPLOT({...},{...},...,'height',h)

description

[hax,hs]=SLIDERPLOT(...) simple single axis plot with a slider to browse through the data. The function returns the handle of the axes and the slider object. Arguments can be anything that the plot command accepts.

[hax,hs]=SLIDERPLOT({...},{...},...) plots multiple plots in multiple axes. Every argument should be a cell array with valid arguments for the plot command.

[hax,hs]=SLIDERPLOT({...},{...},...,'height',h) sets the relative heights of the multiple axes.
