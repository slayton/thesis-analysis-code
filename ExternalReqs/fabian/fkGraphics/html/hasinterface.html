hasinterface

synopsis

check if object supports interface

syntax

    b=HASINTERFACE(h,interface)

description

b=HASINTERFACE(h,interface) where interface is a structure with 'properties' and 'methods' fields, each of which is a cell array of strings. The functions return true if the object has all properties and methods as defined by the interface
