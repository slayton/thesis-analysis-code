getpolyline

synopsis

let's you define a new polyline

syntax

    p=GETPOLYLINE
    p=GETPOLYLINE(param1,val1,...)

description

p=GETPOLYLINE opens a new figure and axes in which you can select a polyline using the mouse. Left mouse button will add a point, middle mouse button will remove the last point. The right mouse button or hitting 'Enter' will end the polyline selection. Pressing 'Esc' will abort the polyline selection. Pressing 'c' will toggle the closed state of the polyline. Pressing 's' will toggle the spline state of the polyline. The function returns a structure with the nodes of the polyline and the closed and spline flags.

p=GETPOLYLINE(param1,val1,...) additional options. Valid options are:

closed
0/1 initial closed state
axes
handle of parent axes
maxnodes
maximum number of nodes
minnodes
minimum number of nodes
spline
0/1 initial spline state
