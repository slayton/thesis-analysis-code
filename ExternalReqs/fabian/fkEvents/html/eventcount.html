eventcount

synopsis

cumulative count of events

syntax

    fcn=EVENTCOUNT(events)
    c=EVENTCOUNT(events,x)

description

fcn=EVENTCOUNT(events) returns a handle to a function that accepts a single arguments and returns the cumulative count of events up to the value of argument.

c=EVENTCOUNT(events,x) returns the cumulative count of events up to x.
