event2sound

synopsis

create sound wave from event times

syntax

    w=EVENT2SOUND(events)
    w=EVENT2SOUND(spikes,parm1,val1,...)

description

w=EVENT2SOUND(events) construct sound wave from vector of event times.

w=EVENT2SOUND(spikes,parm1,val1,...) specifies additional options. Valid options are:

segment
create sound wave for specified time segment only
fs
desired sampling frequency
slowmotion
slow motion factor
slowkernel
true/false the kernels will be slowed down as well
kernel
'box'/'gauss' or a custom kernel
kernel_width
width/standard deviation of kernel
kernel_fs
sampling frequency of custom kernel
amplitude
vector of kernel amplitudes for each event
