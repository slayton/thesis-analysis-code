psth

synopsis

peri-stimulus time histogram

syntax

    h=PSTH(trigger,events)
    [h,lags]=PSTH(...)
    [h,lags,n]=PSTH(...)
    h=PSTH(trigger,events,parm1,val1,...)

description

h=PSTH(trigger,events) returns the psth for the events, given the in the interval [-1 1] around the trigger events. The number of bins in the histogram is 51. The trigger should be a sorted vector, the events can be either a sorted vector or a cell array of sorted vectors. In the latter case, the result is a array.

[h,lags]=PSTH(...) returns the lags used for the histogram.

[h,lags,n]=PSTH(...) returns the number of valid triggers.

h=PSTH(trigger,events,parm1,val1,...) sets optional parameters. Valid parameters are:

lags
vector of bins for histogram
segments
list of segments
normalization
'coef', 'none', 'rate'
