event2bin

synopsis

count events in bins

syntax

    c=EVENT2BIN(events,bins)
    c=EVENT2BIN(events,bins,parm1,val1,...)

description

c=EVENT2BIN(events,bins) return a matrix of event counts for each bin. The size of the matrix is: #events x #bins. The events argument should be a vector or a cell array of vectors. The bins argument should be a nx2 matrix, in which each row defines the start and end of a bin.

c=EVENT2BIN(events,bins,parm1,val1,...) set optional parameters. Valid parameters are:

method
'count', 'binary', 'rate'
