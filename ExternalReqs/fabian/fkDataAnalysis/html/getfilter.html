getfilter

synopsis

return coefficients for eeg filter

description

Syntax [b, s] = getfilter( Fs, band, method )

Description This function returns the coefficients for a FIR filter. Band specifies the frequency band of interest and can be one of: 'theta', 'ripple', 'slow', 'spindle', 'gamma'. The filter is created using one of three methods: 'win', 'ls' or 'pm'. Optionally, a string containing the filter definition is returned.
