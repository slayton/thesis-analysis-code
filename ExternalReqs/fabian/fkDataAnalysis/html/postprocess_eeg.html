postprocess_eeg

synopsis

clip detection and ICA artefact removal

syntax

    POSTPROCESS_EEG(rootdir,epoch)
    POSTPROCESS_EEG(rootdir,epoch,parm1,val1,...)

description

POSTPROCESS_EEG(rootdir,epoch) for each eeg data file found for the specified epoch, clipped regions are found, ica is performed and the user can inspect the components and indicate those that are artefacts.

POSTPROCESS_EEG(rootdir,epoch,parm1,val1,...) additonal options:

Ica
0/1, perform ica
Clip
0/1, search for clipped regions
Artefacts
0/1, ask user to specify artefacts
Inspect
0/1, let user inspect ica results
Interactive
0/1, for each steps ask user for permission
