T2D_scale

synopsis

apply scaling to transformation matrix

syntax

    t=T2D_SCALE(scale)
    t=T2D_SCALE(scale,t)

description

t=T2D_SCALE(scale) returns a transformation matrix t that represent a scaling by factor scale. Scale can be a scalar (unforma scaling along x and y dimensions) or a two-element vector that specifies the scaling factors for x and y dimensions separately.

t=T2D_SCALE(scale,t) applies scaling to existing transformation matrix.
