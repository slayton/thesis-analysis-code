T2D_rotate

synopsis

apply rotation to transformation matrix

syntax

    t=T2D_ROTATE(phi)
    t=T2D_ROTATE(phi,t)

description

t=T2D_ROTATE(phi) returns the transformation matrix t that represents a rotation by phi radians.

t=T2D_ROTATE(phi,t) applies the rotation to existing transformation matrix.

t=T2D_ROTATION(phi,t,origin) applies rotation around the specified origin (default = [0 0])
