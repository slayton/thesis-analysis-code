T2D_translate

synopsis

apply translation to transformation matrix

syntax

    t=T2D_TRANSLATE(offset)

description

t=T2D_TRANSLATE(offset) returns tranaformation matrix representing a translation. Offset can be a scalar or two element vector specifying the translation in x and y dimensions.

t=T2D_TRANSLATION(offset,t) applies translation to exisiting transformation matrix.
