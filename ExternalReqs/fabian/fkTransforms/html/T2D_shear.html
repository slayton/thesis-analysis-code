T2D_shear

synopsis

apply shear to transformation matrix

syntax

    t=T2D_SHEAR(theta)
    t=T2D_SHEAR(theta,t)

description

t=T2D_SHEAR(theta) returns transformation matrix that represents shearing. Theta can be a scalar or a two element vector specifying the shearing angles in the x and y dimensions.

t=T2D_SHEAR(theta,t) apply shearing to existing transformation matrix.
