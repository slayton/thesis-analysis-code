struct2param

synopsis

convert structure to parameter/value pairs

syntax

    p=STRUCT2PARAM(struct)
    p=STRUCT2PARAM(struct,expand)

description

p=STRUCT2PARAM(struct) returns a cell array of parameter/value pairs based on the structure.

p=STRUCT2PARAM(struct,expand) expands substructures
