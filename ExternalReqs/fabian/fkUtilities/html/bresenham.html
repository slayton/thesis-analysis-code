bresenham

synopsis

bresenham's algorithm for line digitization

syntax

    [x,y]=BRESENHAM(x0,x1,y0,y1)

description

[x,y]=BRESENHAM(x0,x1,y0,y1) Digitizes the line (x0,y0)-(x1,y1) and returns the (integer) x,y image coordinates that represent the line.

example

[cx, cy] = bresenham( 1, 40, 20, 80 );
I = sparse(cy, cx, 1);
imagesc( I );
