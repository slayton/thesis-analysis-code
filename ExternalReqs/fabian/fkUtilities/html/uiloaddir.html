uiloaddir

synopsis

user interface for loaddir function

syntax

    [...]=UILOADDIR(startpath,...)

description

[...]=UILOADDIR(startpath,...) graphical user interface for loaddir function. It will show a directory selection window and call loaddir on the selected directory. The top level directory can be specified by the startpath argument; if omitted it defaults to the current directory.
