loaddir

synopsis

executes a load command for directory

syntax

    [...]=LOADDIR(rootdir)
    [...]=LOADDIR(rootdir,...)

description

[...]=LOADDIR(rootdir) runs the loadme.m file in rootdir and returns the requested outputs of loadme or no outputs if loadme is a script.

[...]=LOADDIR(rootdir,...) additional input arguments are passed to the loadme function.
