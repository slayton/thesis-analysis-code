struct_setdiff

synopsis

returns fields in structure S1 that are not present in S2

syntax

    s=STRUCT_SETDIFF(s1,s2)
    s=STRUCT_SETDIFF(s1,s2,1)

description

s=STRUCT_SETDIFF(s1,s2) returns the fields in s1 that are not present in s2. The function recurses if the same field is present in both structs and the two fields themselves are structs.

s=STRUCT_SETDIFF(s1,s2,1) if a field is a struct in s1, but not a struct in s2 (or vice versa), then the fields are considered to be different and thus the field IS returned in the result.
