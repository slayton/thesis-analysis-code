loadscript

synopsis

load parameters from .m file

syntax

    v=LOADSCRIPT(file)
    v=LOADSCRIPT(rootdir,file)

description

v=LOADSCRIPT(file) this will evaluate the given file and return the variables created by the script. If the script defines a "parent" variable which points to a valid script file, then that parent file is loaded first and combined with the variables defined in the child script.

v=LOADSCRIPT(rootdir,file) this will use the script file located in the given root directory.
