matrixfun

synopsis

perform function on matrix for 1 or more dimensions

syntax

    m=MATRIXFUN(m,function)
    m=MATRIXFUN(m,function,dims)

description

m=MATRIXFUN(m,function) for a n-d matrix m, will perform the specified function along the first dimension. The function argument should be a valid function handle that takes a matrix as the first input and the dimension along which to work as the second input. Functions that do not adhere to this convention should be wrapped into an anonymous function handle (i.e. @(m,d) var(m,1,d))

m=MATRIXFUN(m,function,dims) will perform function on the collective dimensions specified in dims.
