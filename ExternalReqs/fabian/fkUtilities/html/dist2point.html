dist2point

synopsis

calculate distance to point

syntax

    d=DIST2POINT(coordinates,center)

description

d=DIST2POINT(coordinates,center) computes the distance between a list of coordinates and center. Coordinates should be given as a n-by-d matrix, where n is the number of coordinates and d is the number of dimensions. By default center is the origin.
