binpack

synopsis

efficient packing of elements in container

syntax

    i=BINPACK(element_sizes,container_size)

description

i=BINPACK(element_sizes,container_size) returns the indices of the elements whose summed size fits best in a container with specified size. If the number of elements > 8 than not all possible permutations are searched, but rather 10000 random permutations.
