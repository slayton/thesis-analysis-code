cell2str

synopsis

convert cell to string representation

syntax

    string=CELL2STR(cell)

description

string=CELL2STR(cell) converts a cell array to a string representation. The cell array can be recreated by eval(string). This function can handle 2d cell arrys, 2d numeric and logical arrays, character strings (not arrays) and structures.
