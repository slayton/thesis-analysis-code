alpha_gen

synopsis

alpha function generator

syntax

    f=ALPHA_GEN(tau)
    f=ALPHA_GEN(tau,gain)
    f=ALPHA_GEN(tau,gain,offset)

description

f=ALPHA_GEN(tau) returns a function handle that can be used to generate an alpha function with the specified tau parameter, gain=1, and offset=0. Alpha functions can be used to model synaptic input. Tau specifies the time-to-peak, gain is the value at the peak and offset determines the shift along the x-axis. An alpha function has the following form: f(x):gain*((x-offset)/tau)*e^(1-(x-offset)/tau)

f=ALPHA_GEN(tau,gain) uses the specified gain.

f=ALPHA_GEN(tau,gain,offset) uses the specified offset.

example

f=alpha_gen(1,2);
fplot( f, [0 10] );
