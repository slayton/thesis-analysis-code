onoff

synopsis

on/off utility

syntax

    s=ONOFF(b)
    s=ONOFF(b,choices)

description

s=ONOFF(b) returns 'on' when b is true and 'off' otherwise.

s=ONOFF(b,choices) where choices is a two element cell array, returns choices{1} if b is true and choices{2} otherwise.
