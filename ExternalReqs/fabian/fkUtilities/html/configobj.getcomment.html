getcomment

synopsis

get comment associate with configobj or a key

syntax

    [comment,inline]=GETCOMMENT(c)
    [comment,inline]=GETCOMMENT(c,key)

description

[comment,inline]=GETCOMMENT(c) retrieves comment and inline comment of a configobj section.

[comment,inline]=GETCOMMENT(c,key) retrieves comment and inline comment of a key.
