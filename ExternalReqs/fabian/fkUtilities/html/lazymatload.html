lazymatload

synopsis

lazy loading of .mat file variables

syntax

    s=LAZYMATLOAD(filename)

description

s=LAZYMATLOAD(filename) returns a structure with a field for each variable contained in the .mat file filename. The first time a field is accessed, the associated data is loaded from the file and stored in the struct.
