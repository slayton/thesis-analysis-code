loadconfig

synopsis

load configuration

syntax

    c=LOADCONFIG(name)

description

c=LOADCONFIG(name) loads the specified configuration file and returns a config object. The default path for configuration files can be set using setconfigpath. If no default path exists or if the name arguments starts with a '.' then the configuration file will be loaded from the current directory.
