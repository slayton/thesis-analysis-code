dirfun

synopsis

apply function to directories recursively

description

DIRFCN(fcn) calls function for current working directory and every subdirectory in it. The first argument to the function will be the full path of the directory.

dirs=DIRFCN(fcn) returns a list of processed subdirectories.

[dirs,out1,out2,...]=DIRFCN(fcn) returns the outputs of the function.

[...]=DIRFCN(fcn,parm1,val1,...) specifies optional parameters. Valid parameters are:

Path
root directory (default = '.')
Recurse
0/1 recurse into subdirectories
MaxLevel
maximum recursion level (default=3)
RegExp
regular expression for directory selection
FullPath
0/1 return full path
Argin
cell array of extra arguments for the function
