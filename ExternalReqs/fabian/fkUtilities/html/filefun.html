filefun

synopsis

apply function to files recursivley

syntax

    FILEFUN(fcn)
    files=FILEFUN(fcn)
    [...]=FILEFUN(fcn,parm1,val1,...)

description

FILEFUN(fcn) calls function for every file in current working directory. The signature of the function should be: [out1,out2,...]=fcn(path,file,level,...)

files=FILEFUN(fcn) returns a list of processed files.

[files,out1,out2,...]=FILEFCN(fcn) returns the outputs of the function.

[...]=FILEFUN(fcn,parm1,val1,...) specifies optional parameters. Valid parameters are:

Path
root directory (default = '.');
Recurse
0/1 recurse into subdirectories
MaxLevel
maximum recursion level (default = Inf)
Mask
wildcard mask for file listing
RegExp
regular expression(s) for file selection
FullPath
0/1 return filename or full path to file
Argin
cell array of extra arguments for the function
