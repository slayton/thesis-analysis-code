apply2struct

synopsis

apply function to fields in structs

syntax

    [out1,out2,...]=APPLY2STRUCT(fcn,argin,structA,structB,...)

description

[out1,out2,...]=APPLY2STRUCT(fcn,argin,structA,structB,...) applies a function field-wise to a set of structures, which all should have the same fields. Argin should be a cell array with additional, fixed arguments for the function. For each field in the structures, the function is called as: fcn( structA(n), structB(n), ..., argin{:} ).

example

X = struct( 'a', rand(10,1), 'b', rand(10,1) );
[Y, I] = apply2struct( @sort, {1, 'descend'}, X );

example

X = struct( 'a', 1, 'b', [5 10] );
Y = struct( 'a', rand(10,1), 'b', [-3 6] );
Z = apply2struct( @plus, [], X, Y );
