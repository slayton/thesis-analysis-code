exp_gen

synopsis

exponential function generator

syntax

    f=EXP_GEN
    f=EXP_GEN(lambda)
    f=EXP_GEN(lambda, offset)

description

f=EXP_GEN returns a function handle that can be used to generate an exponential function with lambda=1 and offset=0. The exponential function has the following form: f(x):lambda*e^( -lambda*(x-offset) )

f=EXP_GEN(lambda) uses the specified lambda.

f=EXP_GEN(lambda, offset) uses the specified lambda and offset.
