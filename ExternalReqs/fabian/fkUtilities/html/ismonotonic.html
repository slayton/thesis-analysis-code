ismonotonic

synopsis

returns a boolean value indicating whether or not a vector is monotonic

syntax

    b=ISMONOTONIC(x)
    b=ISMONOTONIC(x,1)
    b=ISMONOTONIC(x,0)
    b=ISMONOTONIC(x,[],'increasing')
    b=ISMONOTONIC(x,[],'decreasing')
    b=ISMONOTONIC(x,[],'either')
    b=ISMONOTONIC(x,[],[],dim)

description

b=ISMONOTONIC(x) returns a boolean value indicating whether or not a vector is monotonic. By default, ISMONOTONIC returns true for non-strictly monotonic vectors, and both monotonic increasing and monotonic decreasing vectors. For matrices and N-D arrays, ISMONOTONIC returns a value for each column in x.

b=ISMONOTONIC(x,1) works as above, but only returns true when x is strictly monotonically increasing, or strictly monotonically decreasing.

b=ISMONOTONIC(x,0) works as ISMONOTONIC(x).

b=ISMONOTONIC(x,[],'increasing') works as above, but returns true only when X is monotonically increasing.

b=ISMONOTONIC(x,[],'decreasing') works as above, but returns true only when X is monotonically decreasing.

b=ISMONOTONIC(x,[],'either') works as ISMONOTONIC(x, []).

b=ISMONOTONIC(x,[],[],dim) works as above, but along dimension dim.

NOTE: Third input variable is case insensitive, and partial matching is used, so 'd' would be recognised as 'decreasing' etc..

SEE ALSO: is*

example

x = [1:4; 6:-2:2 3]
ismonotonic(x)
ismonotonic(x, [], 'i')
ismonotonic(x, [], [], 2)

x =
1     2     3     4
6     4     2     3
ans =
1     1     1     1
ans =
1     1     0     0
ans =
1
0
