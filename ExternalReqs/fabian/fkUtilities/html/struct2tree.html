struct2tree

synopsis

show matlab structure or path as tree

syntax

    tree=STRUCT2TREE(s)
    tree=STRUCT2TREE(s,depth)
    tree=STRUCT2TREE(s,depth,indent)
    tree=STRUCT2TREE(s,depth,indent,level)

description

tree=STRUCT2TREE(s) returns a character array representing the tree structure of a path or matlab struct.

tree=STRUCT2TREE(s,depth) sets the deepest level of the struct or path that will be returned (defaulr = Inf)

tree=STRUCT2TREE(s,depth,indent) sets the amount of indentation.

tree=STRUCT2TREE(s,depth,indent,level) sets the current level in the tree (used internally by struct2tree).
