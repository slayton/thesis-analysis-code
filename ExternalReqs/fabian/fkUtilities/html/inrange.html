inrange

synopsis

test whether numbers are within range

syntax

    b=INRANGE(x,range)
    b=INRANGE(x,range,'imag')

description

b=INRANGE(x,range) returns true for values in x that are within the specified range. The range is a two element vector specifying the lower and upper bounds of the range. if range(2)<range(1) then the valid range is inverted. If range is scalar then the range will be set to [range Inf].

b=INRANGE(x,range,'imag') returns true if both magnitude and angle are within range.

example

b=inrange(10, [5 9]); %returns 0
b=inrange(10, [9 5]); %returns 1
b=inrange(1+8i, [0 pi]); %returns 1
