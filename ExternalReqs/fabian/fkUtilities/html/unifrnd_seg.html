unifrnd_seg

synopsis

uniformly sampled random numbers from a set of segments

syntax

    r=UNIFRND_SEG(segs)
    r=UNIFRND_SEG(segs,m,n,...)
    r=UNIFRND_SEG(segs,[m,n,...])

description

r=UNIFRND_SEG(segs) returns a number sampled randomly from a set of segments. Segs is a nx2 matrix.

r=UNIFRND_SEG(segs,m,n,...) returns an array of size [m,n,...] with randomly sampled numbers from the segments.

r=UNIFRND_SEG(segs,[m,n,...]) same as above
