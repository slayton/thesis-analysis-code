selectfields

synopsis

select fields from structure

syntax

    s=SELECTFIELDS(s,fields)

description

s=SELECTFIELDS(s,fields) select the fields from the structure s.

example

s = struct('a', 1, 'b', 2, 'c', 3);
s = selectfields(s, {'b', 'c'} );
