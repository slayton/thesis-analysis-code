explorestruct

synopsis

visually explore struct

syntax

    h=EXPLORESTRUCT
    h=EXPLORESTRUCT(dirname)
    h=EXPLORESTRUCT(...,'RootLabel',label)

description

h=EXPLORESTRUCT displays the current directory as a browseable tree in a new figure window and returns the handle of that figure.

h=EXPLORESTRUCT(dirname) displays a tree view of the specified directory

h=EXPLORESRUCT(struct) displays a tree view of the specified matlab structure.

h=EXPLORESTRUCT(...,'RootLabel',label) sets string for the root label.
