super

synopsis

super object of object

syntax

    superobj=SUPER(obj)
    superobj=SUPER(obj,1)

description

superobj=SUPER(obj) returns the super object of an object or [] if none exists. Only works for single inheritance.

superobj=SUPER(obj,1) returns the super base object of an object. Only works for single inheritance.
