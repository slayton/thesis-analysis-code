localmaximum

synopsis

find local maxima

syntax

    peak_i=LOCALMAXIMUM(x)
    peak_i=LOCALMAXIMUM(x,'method', 'gradient')
    peak_t=LOCALMAXIMUM(t,x,...)

description

peak_i=LOCALMAXIMUM(x) returns indices of local maxima in the vector x.

peak_i=LOCALMAXIMUM(x,'method', 'gradient') uses a method that detects zerocrossings in the gradient of x and returns the (interpolated) indices of the local maxima.

peak_t=LOCALMAXIMUM(t,x,...) returns (interpolated) time values from vector t instead of indices.
