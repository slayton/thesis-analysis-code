ceilto

synopsis

round towards Inf

syntax

    m=CEILTO(m)
    m=CEILTO(m, f)

description

m=CEILTO(m) round m to nearest larger integer.

m=CEILTO(m, f) round m to nearest larger multiple of f.

example

ceilto( [3.4 4.8] ) %returns [4 5]
ceilto( [3.3 5.7], 0.25 ) %returns [3.5 5.75]
