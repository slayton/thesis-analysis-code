normalize

synopsis

normalizes an array

syntax

    m=NORMALIZE(m)
    m=NORMALIZE(m,dimension)
    m=NORMALIZE(m,dimension,option)
    m=NORMALIZE(m,dim,option,1)

description

m=NORMALIZE(m) normalize matrix m to the sum along the first dimension.

m=NORMALIZE(m,dimension) normalize along the specified dimension(s). If dimension=0, then the matrix will be normalized as a whole. If dimensions is a vector, then normalization will occur within the subspace defined by those dimensions (e.g. for a 3-d matrix, setting dimension to [1 2] will normalize each x,y plane)

m=NORMALIZE(m,dimension,option) normalize using the specified option. Valid options are:

sum
normalize to the sum
area
normalize to the area (i.e. sum of absolute values)
mean
normalize to mean
max
normalize to max
min
normalize to min
extreme
normalize to extreme value
amp
normalize to maximum-minimum
range
subtract minimum and normalize to maximum-minimum
zscore
subtract mean and normalize to standard deviation
detrend
subtract mean

m=NORMALIZE(m,dim,option,1) ignores NaNs
