randdist

synopsis

will compute the randomization distribution

syntax

    d=RANDDIST(fcn,data)
    d=RANDDIST(fcn,data,n)
    d=RANDDIST(fcn,data,n,argin)
    d=RANDDIST(fcn,data,n,argin,permfcn)

description

d=RANDDIST(fcn,data) returns the randomization distribution by calling the function fcn 1000 times with random row permutations of data. The function should return a scalar.

d=RANDDIST(fcn,data,n) performs n permutations.

d=RANDDIST(fcn,data,n,argin) provides a cell array of extra arguments to the function.

d=RANDDIST(fcn,data,n,argin,permfcn) uses a custom permutation function that takes data as input and returns a permuted (randomized) version of data.
