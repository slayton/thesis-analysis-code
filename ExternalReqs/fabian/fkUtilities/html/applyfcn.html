applyfcn

synopsis

apply function to cell array or all fields of structure

description

Syntax result = applyfcn( fcn, argin, A, B, ... )

Description Applies a function to all cells in a set of cell arrays or all fields in a set of structs. The function fcn is called repetitively for each cell or field and should have the following signature: output = fcn( A_cell_n, B_cell_n, ..., argin1, argin2, ...) output = fcn( A_field_n, B_field_n, ..., argin1, argin2, ...) Thus all of the cell arrays should have the same dimensions and size, likewise, all structs should have the same fields.
