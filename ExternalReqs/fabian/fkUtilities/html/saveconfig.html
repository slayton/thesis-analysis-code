saveconfig

synopsis

save configuration

syntax

    SAVECONFIG(filename, cfg)

description

SAVECONFIG(filename, cfg) saves a configuration object or structure to a file. The default path for configuration files can be set using setconfigpath. If no default path exists or if the name arguments starts with a '.' then the configuration file will be saved in the current directory.
