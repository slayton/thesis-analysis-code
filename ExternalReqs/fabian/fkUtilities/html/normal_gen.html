normal_gen

synopsis

normal function generator

syntax

    f=NORMAL_GEN
    f=NORMAL_GEN(mu)
    f=NORMAL_GEN(mu,sigma)

description

f=NORMAL_GEN returns the handle of a function that can be used to generate a normal distribution with mu=0 and sigma=1.

f=NORMAL_GEN(mu) uses specified mu.

f=NORMAL_GEN(mu,sigma) uses specified mu and sigma,
