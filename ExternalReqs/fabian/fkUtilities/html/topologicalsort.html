topologicalsort

synopsis

topological sort of connection matrix

syntax

    v=TOPOLOGICALSORT(m)

description

v=TOPOLOGICALSORT(m) where m is a square matrix with each element m(x,y) indicating if an edge is present from vertex x to vertex y. The funciton returns the sorted connectivity matrix v.
