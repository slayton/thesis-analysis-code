floorto

synopsis

round towards -Inf

syntax

    m=FLOORTO(m)
    m=FLOORTO(m, f)

description

m=FLOORTO(m) round m to nearest smaller integer.

m=FLOORTO(m, f) round m to nearest smaller multiple of f.

example

floorto( [3.4 4.8] ) %returns [3 4]
floorto( [3.3 5.7], 0.25 ) %returns [3.25 5.5]
