ndmedian

synopsis

compute N-dimensional median

syntax

    m=NDMEDIAN(x)
    m=NDMEDIAN(x,weights)
    m=NDMEDIAN(x,weights, 1)

description

m=NDMEDIAN(x) computes the median of the nxd matrix x, where n is the number of coordinates and d is the number of dimensions. This function minimizes the mean absolute deviation, E(|x-m|). The function returns a local minimum, which is not guaranteed te be the global minimum.

m=NDMEDIAN(x,weights) computes weighted median

m=NDMEDIAN(x,weights, 1) returns the point in x closest to the true median
