save_props

synopsis

save property data

syntax

    SAVE_PROPS(rootdir,target,props)
    SAVE_PROPS(rootdir,target,props,merge)

description

SAVE_PROPS(rootdir,target,props) saves the structure props in the file rootdir/target.props.

SAVE_PROPS(rootdir,target,props,merge) if merge is 1, the function will merge the structure with the data already present in an existing file.
