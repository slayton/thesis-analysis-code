for_each_epoch

synopsis

execute function for each epoch

syntax

    result=FOR_EACH_EPOCH(rootdir,fcn)

description

result=FOR_EACH_EPOCH(rootdir,fcn) executes the specified function for each epoch defined in rootdir/epochs/epochs.def. The signature of the function is: result=fcn(epoch_info, ...), where epoch_info is a structure with the name and start and end times of the epoch. The output is a cell array with the results of the function for each epoch.
