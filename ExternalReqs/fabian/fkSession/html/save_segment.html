save_segment

synopsis

save segment data to mwl-style file

syntax

    SAVE_SEGMENT(rootdir,segment_name,segments,description)

description

SAVE_SEGMENT(rootdir,segment_name,segments,description) save a list of segments to a mwl type file. The file will be stored in rootdir with the name segment_name (extension is .segment). Description is an optional string to give more information about the segments.
