load_props

synopsis

load property data

syntax

    props=LOAD_PROPS(rootdir,target)
    props=LOAD_PROPS(rootdir,target,'var1','var2',...)
    props=LOAD_PROPS(rootdir,target,{'var1','var2',...})
    propnames=LOAD_PROPS(rootdir,target,0)

description

props=LOAD_PROPS(rootdir,target) returns a structure with all variables contained in the property data file rootdir/target.props

props=LOAD_PROPS(rootdir,target,'var1','var2',...) loads only the selected variables from property data file.

props=LOAD_PROPS(rootdir,target,{'var1','var2',...}) loads only the selected variables from property data file.

propnames=LOAD_PROPS(rootdir,target,0) returns a list of the names of all variables in the property data file.
