load_event

synopsis

load event vector from mwl-style file

syntax

    event=LOAD_EVENT(rootdir,event_name)

description

event=LOAD_EVENT(rootdir,event_name) load a previously saved event file with the name event_name from rootdir. The function returns a structure with 'timestamp' and 'description' fields.
