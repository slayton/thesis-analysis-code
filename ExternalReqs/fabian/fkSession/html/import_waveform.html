import_waveform

synopsis

import spike waveform

syntax

    clusters=IMPORT_WAVEFORM(rootdir,clusters,tetrodes)

description

clusters=IMPORT_WAVEFORM(rootdir,clusters,tetrodes) import spike waveform means (calculate if necessary) and waveform properties for all clusters. Waveform values are converted to mV.
