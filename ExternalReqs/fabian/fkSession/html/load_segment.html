load_segment

synopsis

load segment data from mwl-style file

syntax

    segment=LOAD_SEGMENT(rootdir,segment_name)

description

segment=LOAD_SEGMENT(rootdir,segment_name) load a previously saved segment file with the name segment_name from root_dir. The function returns a structure with 'timestamp' and 'description' fields.
