has_props

synopsis

checks if properties exist

syntax

    b=HAS_PROPS(rootdir,target)
    b=HAS_PROPS(rootdir,target,'var1','var2',...)
    b=HAS_PROPS(rootdir,target,{'var1','var2',...})

description

b=HAS_PROPS(rootdir,target) returns true if property data file exists and false otherwise.

b=HAS_PROPS(rootdir,target,'var1','var2',...) returns true for each variable that is present in the property data file contains and false otherwise.

b=HAS_PROPS(rootdir,target,{'var1','var2',...}) returns true for each variable that is present in the property data file contains and false otherwise.
