ratemap_kde

synopsis

compute kernel density estimate of rate map

syntax

    f=RATEMAP_KDE(x,x_ref)
    f=RATEMAP_KDE(...,parm1,val1,...)

description

f=RATEMAP_KDE(x,x_ref) returns a function that will compute the ratemap as a density based on a set of coordinates x and a population of coordinates x_ref. (i.e. spike behavior and total behavior). Only 1d case has been implemented.

f=RATEMAP_KDE(...,parm1,val1,...) uses specified options. Valid options are:

BandWidth
band width for kde estimation (default=1)
SampleFreq
sampling frequency (default=30)
