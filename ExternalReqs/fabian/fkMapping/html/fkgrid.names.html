fkgrid reference pages  

names

synopsis

return grid dimension names

syntax

    n=NAMES(grid)
    n=NAMES(grid,dim)

description

n=NAMES(grid) returns the names of all dimensions

n=NAMES(grid,dim) returns the name of dimension dim

example

grid = fkgrid({'linear', 1:100, 'test'), 1:100);
n=names(grid);
