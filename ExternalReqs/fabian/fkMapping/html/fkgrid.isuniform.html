isuniform

synopsis

test whether grid is uniform

syntax

    b=ISUNIFORM(grid)
    b=ISUNIFORM(grid,dim)

description

b=ISUNIFORM(grid) returns 1 for each grid dimension that is uniform (i.e. uniform spacing and size of bins) or 0 for those that are not. This method is not implemented for circular grids.

b=ISUNIFORM(grid,dim) performs test only on dimension dim

example

grid = fkgrid('linear', [0:100], 'linear', [0 10 30 70 100]);
b=isuniform(grid);
