compoundfilter

synopsis

create a compound filter function

syntax

    f=COMPOUNDFILTER(filter1,[op,]filter2,...)

description

f=COMPOUNDFILTER(filter1,[op,]filter2,...) returns a function that will filter its inputs by sequentially applying the the specified filters. Each pair of filters can (but doesn't have to) be connected by one of the following logical operations: 'and', 'or', 'xor', 'not'. The default logical operation is 'or'.
