fkgrid reference pages 

size

synopsis

get number of bins

syntax

    n=SIZE(grid)
    n=SIZE(grid, dim)

description

n=SIZE(grid) returns the number of bins in each dimension

n=SIZE(grid, dim) returns the number of bins only for dimension dim

example

grid = fkgrid(1:100,1:50)
n = size( grid );
