edges

synopsis

get bin edges

syntax

    e=EDGES(grid)
    b=EDGES(grid,dim)

description

e=EDGES(grid) returns a cell array of vectors with for each dimension the bin edges of the grid.

b=EDGES(grid,dim) returns the bin edges only along dimension dim

example

grid = fkgrid('linear', 0:100);
b = edges(grid);
