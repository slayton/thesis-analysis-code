binsizes

synopsis

bin sizes

syntax

    b=BINSIZES(grid)
    b=BINSIZES(grid,dim)

description

b=BINSIZES(grid) returns a cell array of vectors with for each dimension the bin sizes of the grid.

b=BINSIZES(grid,dim) returns the bin sizes only along dimension dim

example

grid = fkgrid('linear', 0:100);
b = binsize(grid);
