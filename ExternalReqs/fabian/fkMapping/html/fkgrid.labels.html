labels

synopsis

get bin edges labels

syntax

    l=LABELS(grid)
    l=LABELS(grid,dim)
    l=LABELS(grid,dim,bins)

description

l=LABELS(grid) return a cell array of cell array strings containing text representations of the bin edges for each dimension

l=LABELS(grid,dim) return labels only for specified dimension

l=LABELS(grid,dim,bins) return labels only for specified bins

example

grid = fkgrid(1:100);
lbls = labels(grid);
