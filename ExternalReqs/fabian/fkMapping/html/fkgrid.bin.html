bin

synopsis

bin coordinates

syntax

    b=BIN(grid,coords)
    b=BIN(grid,coords,dim)

description

b=BIN(grid,coords) bin the n-D coordinates using the n-D grid. The function returns for each coordinate the bins each of the dimensions is located in.

b=BIN(grid,coords,dim) returns the bin number of the 1-D coordinates along dimension dim.

example

grid = fkgrid('linear', 1:100, 'linear', 0:10);
b = bin( grid, [1 2; 4 8]);
b = bin( grid, [2; 8], 2);
