centers

synopsis

get bin centers

syntax

    c=CENTERS(grid)
    c=CENTERS(grid,dim)

description

c=CENTERS(grid) returns a cell array of vectors with for each dimension the bin centers of the grid.

c=CENTERS(grid,dim) returns the bin centers only along dimension dim

example

grid = fkgrid('linear', 0:100);
c = centers(grid);
