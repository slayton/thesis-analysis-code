filtercoords

synopsis

filter a set of coordinates

syntax

    coords=FILTERCOORDS(coords,filters)
    [coords,idx]=FILTERCOORDS(...)

description

coords=FILTERCOORDS(coords,filters) returns only those coordinates that are retained by the set of filters. The filters argument is a cell array with function handles for each dimension (column) of the coordinates array.

[coords,idx]=FILTERCOORDS(...) also returns the logical index array used for filtering.
